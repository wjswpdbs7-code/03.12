2진수에서 1의보수는 얼마냐면 0은 1로, 1은 0으로 바꿔주는 것입니다. 이를 이해하기 위해서 쉽게 문학적 비유하자면, 1의 보수는 마치 거울에 비친 모습과 같습니다. 예를 들어, 숫자 5를 2진수로 표현하면 00000101이 됩니다. 이 숫자의 1의 보수는 11111010이 됩니다. 이는 마치 거울에 비친 모습처럼, 0은 1로, 1은 0으로 바뀌는 것을 의미합니다. 따라서, 1의 보수는 숫자의 모든 비트를 반전시키는 것을 의미합니다.
숫자를 2진수로 나타내는 방법은 숫자를 2로 나누면서 나머지를 기록하는 것입니다. 예를 들어, 숫자 5를 2진수로 나타내려면, 5를 2로 나누면 몫이 2이고 나머지가 1입니다. 다시 2를 2로 나누면 몫이 1이고 나머지가 0입니다. 마지막으로, 1을 2로 나누면 몫이 0이고 나머지가 1입니다. 따라서, 숫자 5를 2진수로 나타내면 00000101이 됩니다. 이 방법을 사용하면, 어떤 숫자든지 2진수로 나타낼 수 있습니다. 0부터 10까지 나열하면 다음과 같습니다:

|는 or 비트 연산자. or 연산자는 하나라도 true일 때 true를 반환하는 연산자입니다.

<<,>>는 시프트 연산자. 시프트 연산자는 비트를 왼쪽이나 오른쪽으로 이동시키는 연산자입니다. 

^는 xor 비트 연산자. xor 연산자는 두 비트가 다를 때 true를 반환하는 연산자입니다.

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    int min = 1; // 최소값
    int max = 100; // 최대값
    int random_number;

    // 난수 생성을 위한 시드 설정
    srand(time(0));

    // min과 max 사이의 랜덤한 숫자 생성
    random_number = rand() % (max - min + 1) + min;

    printf("생성된 난수: %d\n", random_number);
    return 0;
}

#### 정렬 sorting: 데이터를 일정한 순서로 나열하는 과정입니다. 예시:
#include <stdio.h>

void bubble_sort(int arr[], int n) 
{
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                // 두 요소를 교환
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

전역변수 지역변수 차이와 작성법
전역변수(global variable): 프로그램 전체에서 접근할 수 있는 변수입니다. 함수 외부에 선언됩니다. 예시:
#include <stdio.h>
int global_variable = 10; // 전역변수 선언
void function() {
    printf("전역변수의 값: %d\n", global_variable); // 전역변수에 접근
}
지역변수(local variable): 특정 함수 내에서만 접근할 수 있는 변수입니다. 함수 내부에 선언됩니다. 예시:
#include <stdio.h>
void function() {
    int local_variable = 20; // 지역변수 선언
    printf("지역변수의 값: %d\n", local_variable); // 지역변수에 접근
}
지역변수는 함수가 호출될 때마다 새롭게 생성되고, 함수가 종료되면 소멸됩니다. 반면에 전역변수는 프로그램이 실행되는 동안 계속 존재하며, 모든 함수에서 접근할 수 있습니다. 따라서, 전역변수는 프로그램 전체에서 공유되는 데이터를 저장하는 데 사용되고, 지역변수는 특정 함수 내에서만 필요한 데이터를 저장하는 데 사용됩니다. 전역변수와 지역변수는 각각의 용도에 맞게 적절히 사용되어야 하며, 전역변수는 프로그램의 복잡성을 증가시킬 수 있으므로 가능한 한 지역변수를 사용하는 것이 좋습니다.
